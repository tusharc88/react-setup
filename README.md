# React Starter kit

### How to Start
```
git clone https://github.com/tusharc88/react-setup.git
cd react-setup
npm install
```
### Important commands

###### To start application on port 3000
```
npm run start
```

###### To bundle application into js file
```
npm run build
```

###### To run tests
```
npm run test
```

###### To watch for tests
```
npm run test:watch
```
